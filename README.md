# Capstone

## 사용 스택
- **Frontend**: React + TypeScript
- **Rich Text Editor**: Quill
- **Styling**: CSS, HTML
- **Build Tool**: Vite

## 실행 방법
- 의존성 설치 후 개발 서버를 실행합니다.

```bash
npm install
npm run dev
```

- Vite 개발 서버가 실행되며 브라우저에서 안내된 주소로 접속하면 앱이 동작합니다.
